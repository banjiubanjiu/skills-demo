◆ 项目 Skills 展示

★ 核心

Git Auto-Commit

自动化完整的 git 提交流程：暂存所有更改、生成智能提交消息、创建提交并推送到远程仓库。

  • 自动检测 git 仓库和更改
  • 基于文件操作生成智能提交消息
  • 一键完成暂存、提交和推送
  • 支持中英文触发词
提交代码 commit 自动提交 push

Skill Creator

创建有效技能的完整指南。帮助你理解 Skills 的核心原则、设计模式和最佳实践。

  • 渐进式信息披露设计原则
  • Skill 结构和命名规范
  • 脚本、引用和资产的组织方式
  • 完整的创建流程指导
创建新技能 更新技能 学习技能设计

Skill Installer

从精选列表或 GitHub 仓库安装 Skills。自动检测系统类型（Claude Code 或 Codex）。

  • 列出精选的可用 skills
  • 从 GitHub 仓库安装技能
  • 支持私有仓库访问
  • 自动目标系统检测
安装技能 列出可用技能 GitHub

Skill Writer

指导用户创建符合最佳实践的 Agent Skills，帮助编写 SKILL.md 文件和设计技能结构。

  • 步骤化的技能创建流程
  • YAML frontmatter 编写指南
  • 有效的描述编写技巧
  • 验证和调试方法
编写技能 设计技能结构 SKILL.md

◆ 如何使用 Skills

01

发现 Skill

根据你的需求选择合适的 Skill，查看描述和触发词

02

安装 Skill

将 Skill 放置在 ~/.claude/skills/ 或项目的 .claude/skills/ 目录

03

重启 Claude

重启 Claude Code 以加载新的 Skill

04

开始使用

使用触发词或描述相关任务，Claude 会自动激活相应的 Skill

◆ 为什么使用 Skills

🎯

上下文高效

渐进式加载机制，只在需要时加载相关内容，最大化上下文窗口利用率

🔄

可重用性

一次编写，多次使用。脚本和模板可以避免重复编写相同代码

📈

团队协作

项目 Skills 可以提交到 git，团队成员共享统一的工作流程和规范

🛡️

可靠性

对于关键任务，使用脚本确保确定性的执行结果
